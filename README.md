## Cómo abrir el proyecto en Unity

### 1. Clonar el repositorio

Abre una terminal y ejecuta:

```bash
git clone https://github.com/Mati2F/motodelivery.git
```

### 2. Abrir Unity Hub

### 3. Agregar el proyecto
* Haz clic en "Add" / "Agregar proyecto"
* Selecciona la carpeta clonada (motodelivery)

### 4. Abrir el proyecto
* Selecciona el proyecto en Unity Hub
* Haz clic en "Open" / "Abrir"

## Para abrir la escena actual:
> [!IMPORTANT]
__Dentro de Assets, ir a la carpeta "Scenes" y apretar el icono del cubo llamado "SimplePoly City - Low Poly Assets_Demo Scene.unity"__
